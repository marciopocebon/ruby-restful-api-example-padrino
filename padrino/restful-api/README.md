## Getting Started:

pre-requisites:
- ruby 2.2.3
- bundler
- sqlite

### install dependencies
```shell
$ bundle install
```

### run the migrations
```shell
$ padrino ar:migrate
```

### run unit tests
```shell
$ rspec
```

# start the app
```shell
$ ./bin/restful-api
```
